## ストーリー
マゼンタカラーの謎の生物が通りがかったとある村で、ヒヨコちゃんの救出を要請された。<br>
はたして謎の生物は、村の先にあるという洞窟の主から、ヒヨコちゃんを助け出すことができるだろうか！？

## 操作方法
#### カーソルキー左右
謎の生物（自機）を左右に動かす
#### スペースキー
ジャンプする
#### Bキー
ファイヤーを発射する
#### ゲームパッドでも操作できます
